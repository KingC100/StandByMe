# StandByMe
JavaFXで書いた、
ポケモン第六世代(ORAS対応済)の対戦補助ツール。

入力されたポケモン名に応じて
アイコン、基礎情報表示と素早さの計算、
タイプ相性表の表示を行います。

ショートカットキーで対戦考察wiki, wiki, 図鑑で検索。

![image_alt_text](http://f.st-hatena.com/images/fotolife/K/KingCrimson96/20150628/20150628201955.gif?1435490607?raw=true)
